About PoGo Stats

A personal Discord bot + dashboard for tracking Pokemon Go catch and raid stats from PolygonX webhook messages. Built for personal, non-commercial use.

Credits

  • FastAPI - the backend web framework
  • Uvicorn - the ASGI server FastAPI runs on
  • discord.py - the Discord bot library
  • httpx - used to fetch and cache Pokemon sprites
  • python-dotenv - loads the bot token from .env
  • Chart.js - the dashboard and raid charts
  • Leaflet.js - the interactive maps
  • Leaflet.heat - the catch density heatmap
  • CARTO - the free dark map tiles
  • OpenStreetMap contributors - the underlying map data
  • PokeAPI/sprites - the Pokemon sprite images
  • PolygonX - the Discord webhook source this tool reads catch/raid data from
